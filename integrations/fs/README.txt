These demonstrate the usage of the FormSpring API using PHP.  It has only been tested with PHP 5, and requires that you have the CURL library (http://php.net/curl) enabled in your installation.

Feel free to copy and paste into your application.  All code is licensed under the MIT license http://www.opensource.org/licenses/mit-license.php.

---

formspring.php - library that's used to make requests to the API

data.php - example code that paginates through data submitted on a form

edit.php - example code that submits a new entry to a form

forms.php - example code that lists info about forms accessible via the API

submit.php - example code that edits an existing submission
